# Vision

Lantern Engine should prove one thing first: **controlling light must feel distinct, readable, and tactically satisfying**.

For this repository's first release, the success bar is intentionally modest:

- moving around the arena feels responsive
- the beam is readable in motion
- at least one bounce/refraction moment feels clever
- enemy pressure forces repositioning
- one reward choice visibly changes later combat

This repo is therefore not trying to prove full roguelite breadth. It is trying to prove the core promise behind Prism.
