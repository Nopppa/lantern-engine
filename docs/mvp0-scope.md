# MVP-0 Scope

## Included

- one authored arena
- player movement
- HP + Energy resource
- one Prism primary attack: Refraction Beam
- one simple Prism setup skill: Prism Node
- at least one bounce/refraction interaction
- two enemy archetypes
- basic hit/death feedback
- one reward selection step
- retry / restart loop
- dev shortcuts
- repo documentation for continuation

## Excluded on purpose

- procedural generation
- full scene/component decomposition
- broad data-driven authoring pipeline
- save/load
- meta progression
- multiple schools
- broad upgrade pool
- art/audio polish pass
- boss/miniboss
- localization
- platform options UI

## Acceptance test for this milestone

A player should be able to:

1. launch the build
2. move immediately
3. cast the beam with visible feedback
4. kill enemies
5. create a bounce or prism redirection moment
6. finish at least one encounter
7. choose one upgrade
8. continue or restart the run without reopening the app
